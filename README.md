# Heart Disease Prediction using Machine Learning

This project applies Machine Learning algorithms to predict the likelihood of heart disease using patient medical data.

## Dataset
The project uses the Cleveland Heart Disease dataset containing 303 patient records and 13 medical features.

## Machine Learning Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Random Forest
- Artificial Neural Network (MLP)

## Results
The Support Vector Machine (SVM) achieved the best performance with an accuracy of 90%, followed by KNN with 88.33%.

## Visualizations
- Model comparison graph
- Confusion matrix
- ROC curve

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

This project demonstrates the use of Machine Learning techniques for healthcare data analysis and predictive modeling.
